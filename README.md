# DailyLog - Agile & DevOps Vlog Platform

**DailyLog** is a full-stack vlogging platform demonstrating Agile & DevOps practices. It allows users to post daily updates, "like" content, and engage via threaded comments, backed by a fully containerized architecture.

## Features
* **Vlogging:** Create and view posts with video links.
* **Engagement:** Real-time "Likes" and threaded Comments (One-to-Many).
* **Persistence:** PostgreSQL database ensures data survival.
* **DevOps:** CI/CD via GitHub Actions and Dockerized infrastructure.

## Tech Stack
* **Frontend:** Next.js (React, TypeScript)
* **Backend:** Python (FastAPI, SQLAlchemy)
* **Database:** PostgreSQL (Prod) / SQLite (Test)
* **Infrastructure:** Docker & GitHub Actions
